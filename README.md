# Plant Sequencing Database

A centralized database project for integrating raw sequencing data from the European Nucleotide Archive (ENA) with structured biological annotations.

## Overview
This project maps large-scale FASTQ sequences to JSON-based metadata fields such as:
- `trophic_level`
- `observed_biotic_relationship`
- `known_pathogenicity`

## Initialization
- Project initialized.

- Bioagents Memory Graph established with core schema nodes.
